# AI & Development Project Portfolio – Pragyan Prayas Jena

This repository contains a collection of my projects in Machine Learning, Deep Learning, and Web Development. Each project focuses on practical implementation, covering the complete workflow from data processing and model development to deployment and user interface design.

## Projects

### Pneumonia Detection using CNN
Deep learning model built with TensorFlow/Keras to detect pneumonia from chest X-ray images. Includes image preprocessing, model training, evaluation, and a Flask-based interface for real-time prediction.

### Cat vs Dog Classification
Image classification system using multiple machine learning models including SVM, Random Forest, and Logistic Regression. Integrated with a Flask web interface for user interaction.

### Stock Price Prediction
Time-series forecasting project using machine learning techniques to predict future stock trends. Includes data preprocessing, feature engineering, and performance analysis.

### Spotify Clone
Responsive frontend music player interface built using HTML and CSS, focusing on layout design and user experience.
